bootstrap
=========

Rebuild (or update) my home directory skeleton.


Deploy:
-------

**Prereqs:**

 * git
 * A POSIX shell

**Procedure:**

Clone repo to your home directory:

    git clone git://github.com/nurfherder/bootstrap.git ~/code/github.com/nurfherder/bootstrap

Install into ~/bin:

    cd ~/code/bin/bootstrap
    ./bootstrap_home.sh
